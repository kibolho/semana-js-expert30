## :zap: Installation
Run the commands below to install and run the app.
  ```sh
    #Clone the project
    git clone https://github.com/kibolho/semana-js-expert30
  ```

  ```sh
    #Go to server folder
    cd server

    #Install dependencies
    yarn

    #Start the server
    yarn dev
   ```

     ```sh
    #Go to client folder
    cd client

    #Install dependencies
    yarn

    #Start the User01
    yarn user01

    #Open a new terminal
    
    #Start the User02
    yarn user02
   ```
